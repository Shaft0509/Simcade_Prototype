# Simcade_Prototype

The Drove
Overview

The Drove is a physics-based car racing game built in Unity. The player drives a car through a highway to reach a goal before the timer runs out. The game features realistic car physics with front-wheel-style turning, acceleration boost, and interactive UI for win/lose states.

Controls

W / S → Move forward / backward

A / D → Turn left / right (rotates around front wheels)

Left Shift / Right Shift → Boost acceleration

Objective

Reach the goal object in the scene before the timer expires.

If the player reaches the goal in time → You Win!

If time runs out → You Lose!
